# Grind 75

## Overview of Grind 75

Grind 75 is a curated list of 75 essential coding interview questions designed to help you prepare for technical interviews at top technology companies. Created by the author of Blind 75, this list is structured to provide a comprehensive yet focused approach to interview preparation.

This repository contains my solutions and notes for each problem, organized by week and topic for systematic learning.

## Topics Covered

- **Arrays**: Fundamental array operations and algorithms
- **Strings**: String manipulation and pattern matching
- **Hashing**: Hash tables and set operations
- **Two Pointers**: Efficient array/string traversal techniques
- **Sliding Window**: Subarray/substring optimization problems
- **Stack/Queue**: LIFO and FIFO data structure applications
- **Linked List**: Linear data structure operations
- **Trees**: Binary trees, BST, and tree traversals
- **Graphs**: Graph traversal and pathfinding algorithms
- **Heaps/Priority Queue**: Priority-based data operations
- **Binary Search**: Efficient searching in sorted data
- **Dynamic Programming**: Optimization through memoization
- **Math**: Mathematical concepts and number theory
